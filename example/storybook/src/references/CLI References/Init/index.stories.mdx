# init

This command initialises a project with `bolt.yaml` file & `.boltignore` file.

```bash
$ bolt init --help
Usage: bolt init [options]

Inits the project with bolt.yaml file

Options:
  -n, --name <name>  Project name
  -h, --help         display help for command
```
