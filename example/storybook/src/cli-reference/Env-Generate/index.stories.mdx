# env:generate

This command generates .env file from .env.tpl file available in each bolt service.

```bash
$ bolt env:generate --help
Usage: bolt env:generate [options]

Generates ".env" for all the services from ".env.tpl" file in each service

Options:
  -h, --help  display help for command
```
